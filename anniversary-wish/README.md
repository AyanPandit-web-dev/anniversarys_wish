# Anniversary Wish Project

Welcome to the **Anniversary Wish** project! This web application is designed to create a delightful and engaging experience for sending anniversary wishes. With vibrant animations and a user-friendly interface, it aims to make every wish special.

## Features

- **Responsive Design**: Optimized for mobile devices, ensuring a seamless experience on any screen size.
- **Vibrant Colors**: Enhanced color palette for a visually appealing interface.
- **Smooth Animations**: Engaging animations that bring the wishes to life.
- **Interactive Elements**: User-friendly buttons and modals for easy navigation and interaction.
- **Confetti Effects**: Celebrate special moments with delightful confetti animations.

## Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd anniversary-wish
   ```

2. **Open `index.html`**: Launch the main HTML file in your web browser to view the application.

3. **Customize Styles**: Modify the CSS files in the `src/css` directory to personalize the look and feel.

## Usage

- Click the "Wish" button to open the modal and send your heartfelt message.
- Enjoy the confetti effects and animations that enhance the experience.
